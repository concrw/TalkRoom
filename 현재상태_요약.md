## 프로젝트 개요

BMIC와 TALKROOM 프로젝트를 성공적으로 통합하여, TALKROOM이라는 이름으로 실행 중심 북클럽 플랫폼을 개발 중입니다.

### 핵심 기능
- 실시간 음성 토크룸
- 체계적인 훈련 시스템 (5가지 템플릿)
- 일일 기록 (아침 다짐, 저녁 리뷰)
- 커뮤니티 피드
- 리뷰 게이팅 시스템

## 데이터베이스 스키마

### 12개 테이블

| 테이블 | 행 | 설명 |
|--------|-----|------|
| users | Profile data | 사용자 프로필 (name, level, streak_days) |
| talk_rooms | Room info | 토크룸 (title, description, capacity, price) |
| room_participants | Join records | 참여자 (user_id, room_id, status) |
| room_favorites | Favorites | 관심 토크룸 |
| voice_sessions | Live sessions | 음성 세션 (session_url, status) |
| chat_messages | Chat logs | 실시간 채팅 |
| training_courses | Course plans | 훈련 코스 (template_type, total_days) |
| daily_logs | Daily records | 일일 기록 (morning_promise, evening_review) |
| feed_posts | Community posts | 커뮤니티 포스트 |
| post_likes | Like records | 좋아요 |
| reviews | Room reviews | 리뷰 |
| notifications | Notifications | 알림 |
