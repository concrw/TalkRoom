## 2. Supabase 데이터베이스 설정

### 마이그레이션 적용 방법

#### 방법 1: Supabase Dashboard 사용

1. https://supabase.com 로그인
2. 프로젝트 선택
3. Database > SQL Editor 이동
4. `supabase/migrations/` 폴더의 파일을 순서대로 실행
   - 파일명의 타임스탬프 순서대로 실행
   - 총 27개 파일

#### 방법 2: Supabase CLI 사용 (권장)

```bash
# Supabase CLI 설치
npm install -g supabase

# 프로젝트 링크
supabase link --project-ref ctvdsjzazqoodeqenwza

# 마이그레이션 적용
supabase db push
```
